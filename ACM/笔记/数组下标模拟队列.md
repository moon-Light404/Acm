我们可以不采用`queue`来模拟队列完成`bfs`操作

```c++
int hh = 0; // 队头
int tt = -1; // 队尾
int q[N];
头--尾
hh tt
0  -1 开始hh > t b    
    
// 当 hh <= tt 时，表示队列中有元素
hh.....tt
队头	  队尾
// 队尾插入元素
void push(int x)
{
    q[++tt] = x; // 插入的第一个元素为q[0]
}

// 队头删除元素
void pop()
{
 	hh++;
}

// 判断空
bool empty()
{
    if(hh > tt)	return true;
    else return false;
}

// 取出队头元素
int query()
{
    return q[hh];
}
```
